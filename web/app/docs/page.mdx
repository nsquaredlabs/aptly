# What is Aptly?

Aptly is a **compliance-as-a-service middleware** that sits between your application and LLM providers (OpenAI, Anthropic, Google, etc.), automatically handling PII redaction, audit logging, and compliance requirements.

## The Problem

Organizations want to use Large Language Models but face critical challenges:

- **PII Leakage Risk** - Sending customer data to external LLM APIs exposes sensitive information like SSNs, emails, and credit cards
- **Compliance Requirements** - HIPAA, SOC 2, and other frameworks require immutable audit trails of all data processing
- **Multi-Provider Complexity** - Each LLM provider has different APIs, making it hard to switch or use multiple providers
- **Security Overhead** - Building PII detection, redaction, and audit systems from scratch takes months

## The Solution

Aptly solves these problems with a **drop-in OpenAI-compatible proxy** that:

### 🔒 Automatic PII Redaction

Detects and redacts sensitive data before sending to LLMs using Microsoft Presidio:
- **Person names** - "John Smith" → "PERSON_A"
- **Social Security Numbers** - "123-45-6789" → "SSN_A"
- **Email addresses** - "user@company.com" → "EMAIL_A"
- **Credit cards, phone numbers, and more**

Three redaction modes: **mask**, **hash**, or **remove**

### 📋 Immutable Audit Logs

Every request is logged with:
- Full request/response payloads
- PII detections and redactions
- Token usage and costs
- Timestamps and metadata
- Database-enforced immutability (can't be modified or deleted)

### 🔄 Multi-Provider Support

Single unified API works with:
- OpenAI (GPT-4, GPT-3.5)
- Anthropic (Claude)
- Google (Gemini)
- Cohere
- Together AI
- Any provider supported by LiteLLM

## Quick Start

```python
# Before (direct OpenAI)
import openai
response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Hello"}]
)

# After (via Aptly)
import requests
response = requests.post(
    "https://api-aptly.nsquaredlabs.com/v1/chat/completions",
    headers={"Authorization": "Bearer apt_live_..."},
    json={
        "model": "gpt-4",
        "messages": [{"role": "user", "content": "Hello"}],
        "api_keys": {"openai": "sk-..."}
    }
)
```
