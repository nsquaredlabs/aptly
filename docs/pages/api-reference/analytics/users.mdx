---
title: "User Breakdown"
api: "GET /v1/analytics/users"
description: "Get usage breakdown by end-user ID"
---

See which end-users are driving the most API activity. Usage is grouped by the `user` field passed in each [Chat Completions](/api/chat-completions) request. Requests made without a `user` field are collected under `users_with_no_id`.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token with your Aptly API key: `Bearer apt_live_xxx` or `Bearer apt_test_xxx`
</ParamField>

## Query Parameters

<ParamField query="start_date" type="string">
  Start of the analysis period.

  Formats accepted:
  - ISO 8601: `2026-01-01T00:00:00Z`
  - Date only: `2026-01-01`

  Default: 30 days ago
</ParamField>

<ParamField query="end_date" type="string">
  End of the analysis period.

  Formats accepted:
  - ISO 8601: `2026-01-31T23:59:59Z`
  - Date only: `2026-01-31`

  Default: Current time
</ParamField>

<ParamField query="limit" type="integer" default={50}>
  Maximum number of users to return, sorted by request count descending. Clamped to a maximum of `100`.
</ParamField>

## Response

<ResponseField name="users" type="array" required>
  Top users by request count

  <Expandable title="user entry">
    <ResponseField name="user_id" type="string">
      The user identifier passed in the original chat completions request
    </ResponseField>

    <ResponseField name="requests" type="integer">
      Total requests made by this user in the period
    </ResponseField>

    <ResponseField name="tokens_input" type="integer">
      Total input tokens across this user's requests
    </ResponseField>

    <ResponseField name="tokens_output" type="integer">
      Total output tokens across this user's requests
    </ResponseField>

    <ResponseField name="cost_usd" type="number">
      Estimated total cost attributed to this user in USD
    </ResponseField>

    <ResponseField name="last_active" type="string">
      ISO 8601 timestamp of this user's most recent request in the period
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="users_with_no_id" type="object" required>
  Aggregate stats for requests that had no `user` field

  <Expandable title="users_with_no_id">
    <ResponseField name="requests" type="integer">
      Requests with no user ID
    </ResponseField>

    <ResponseField name="tokens_input" type="integer">
      Input tokens for anonymous requests
    </ResponseField>

    <ResponseField name="tokens_output" type="integer">
      Output tokens for anonymous requests
    </ResponseField>

    <ResponseField name="cost_usd" type="number">
      Estimated cost for anonymous requests in USD
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="total_unique_users" type="integer" required>
  Total distinct user IDs in the period. This count is independent of `limit` — use it to determine whether the `users` array has been truncated.
</ResponseField>

<ResponseField name="period_start" type="string" required>
  Effective start date of the period (`YYYY-MM-DD`)
</ResponseField>

<ResponseField name="period_end" type="string" required>
  Effective end date of the period (`YYYY-MM-DD`)
</ResponseField>

## Example Requests

<CodeGroup>

```bash cURL
curl "https://api-aptly.nsquaredlabs.com/v1/analytics/users?limit=10" \
  -H "Authorization: Bearer apt_live_xyz789"
```

```python Python
import requests

response = requests.get(
    "https://api-aptly.nsquaredlabs.com/v1/analytics/users",
    headers={"Authorization": "Bearer apt_live_xyz789"},
    params={"limit": 10}
)

data = response.json()
print(f"Unique users: {data['total_unique_users']}")
print(f"Anonymous requests: {data['users_with_no_id']['requests']}\n")

for user in data["users"]:
    print(f"{user['user_id']} — {user['requests']} requests, ${user['cost_usd']:.2f}")
```

```javascript JavaScript
const response = await fetch(
  'https://api-aptly.nsquaredlabs.com/v1/analytics/users?limit=10',
  { headers: { 'Authorization': 'Bearer apt_live_xyz789' } }
);

const data = await response.json();
console.log(`Unique users: ${data.total_unique_users}`);
data.users.forEach(u => {
  console.log(`${u.user_id}: ${u.requests} requests, $${u.cost_usd}`);
});
```

</CodeGroup>

## Notes

- `total_unique_users` reflects all users in the period regardless of `limit`. Compare it to `users` length to know if the list is truncated.
- Any `limit` value above `100` is clamped to `100`.
- Pass the `user` field in [Chat Completions](/api/chat-completions) to associate requests with a user ID. Without it, usage lands in `users_with_no_id`.
