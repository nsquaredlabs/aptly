---
title: "Query Audit Logs"
api: "GET /v1/logs"
description: "Query audit logs with filtering and pagination for compliance and usage analysis"
---

Retrieve paginated audit logs with optional filtering by date range, user ID, and model. Every request to the chat completions endpoint is automatically logged for compliance.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token with your Aptly API key: `Bearer apt_live_xxx` or `Bearer apt_test_xxx`
</ParamField>

## Query Parameters

<ParamField query="start_date" type="string">
  Start date for filtering logs

  Formats accepted:
  - ISO 8601: `2026-01-01T00:00:00Z`
  - Date only: `2026-01-01` (defaults to 00:00:00 UTC)

  Default: 30 days ago
</ParamField>

<ParamField query="end_date" type="string">
  End date for filtering logs

  Formats accepted:
  - ISO 8601: `2026-01-31T23:59:59Z`
  - Date only: `2026-01-31` (defaults to 23:59:59 UTC)

  Default: Current time
</ParamField>

<ParamField query="user_id" type="string">
  Filter by user ID (from the `user` field in chat completions request)
</ParamField>

<ParamField query="model" type="string">
  Filter by model name (e.g., `gpt-4`, `claude-3-5-sonnet-20241022`)
</ParamField>

<ParamField query="limit" type="integer" default={50}>
  Number of logs per page (1-100)
</ParamField>

<ParamField query="page" type="integer" default={1}>
  Page number (1-indexed)
</ParamField>

## Response

<ResponseField name="logs" type="array" required>
  Array of audit log summary objects

  <Expandable title="log object">

## Use Cases

### Monthly Usage Report

Generate a monthly usage report:

```python
import requests
from datetime import datetime
from collections import defaultdict

def generate_monthly_report(api_key, year, month):
    """Generate monthly usage report."""
    # Calculate date range
    start_date = f"{year}-{month:02d}-01"
    if month == 12:
        end_date = f"{year + 1}-01-01"
    else:
        end_date = f"{year}-{month + 1:02d}-01"

    # Fetch all logs for the month
    all_logs = []
    page = 1

    while True:
        response = requests.get(
            "https://api.aptly.dev/v1/logs",
            headers={"Authorization": f"Bearer {api_key}"},
            params={
                "start_date": start_date,
                "end_date": end_date,
                "limit": 100,
                "page": page
            }
        )
        result = response.json()
        all_logs.extend(result['logs'])

        if page >= result['pagination']['total_pages']:
            break
        page += 1

    # Analyze
    total_requests = len(all_logs)
    total_cost = sum(log['cost_usd'] for log in all_logs)
    total_tokens = sum(log['tokens_input'] + log['tokens_output'] for log in all_logs)

    # Group by model
    by_model = defaultdict(lambda: {"count": 0, "cost": 0, "tokens": 0})
    for log in all_logs:
        model = log['model']
        by_model[model]["count"] += 1
        by_model[model]["cost"] += log['cost_usd']
        by_model[model]["tokens"] += log['tokens_input'] + log['tokens_output']

    # Print report
    print("=" * 60)
    print(f"MONTHLY USAGE REPORT - {year}-{month:02d}")
    print("=" * 60)
    print(f"\nTotal Requests: {total_requests:,}")
    print(f"Total Tokens: {total_tokens:,}")
    print(f"Total Cost: ${total_cost:.2f}")

    print(f"\n--- BY MODEL ---")
    for model, stats in sorted(by_model.items(), key=lambda x: x[1]['cost'], reverse=True):
        print(f"\n{model}:")
        print(f"  Requests: {stats['count']:,}")
        print(f"  Tokens: {stats['tokens']:,}")
        print(f"  Cost: ${stats['cost']:.2f}")

    return {
        "total_requests": total_requests,
        "total_cost": total_cost,
        "total_tokens": total_tokens,
        "by_model": dict(by_model)
    }

# Usage
report = generate_monthly_report("apt_live_xyz789", 2026, 1)
```

### PII Detection Analysis

Analyze PII detection patterns:

```python
def analyze_pii_detection(api_key, days=30):
    """Analyze PII detection in recent logs."""
    from datetime import datetime, timedelta

    end_date = datetime.now()
    start_date = end_date - timedelta(days=days)

    # Fetch logs
    response = requests.get(
        "https://api.aptly.dev/v1/logs",
        headers={"Authorization": f"Bearer {api_key}"},
        params={
            "start_date": start_date.isoformat(),
            "end_date": end_date.isoformat(),
            "limit": 100
        }
    )
    logs = response.json()['logs']

    # Analyze
    total = len(logs)
    input_pii = sum(1 for log in logs if log['pii_detected_input'])
    response_pii = sum(1 for log in logs if log['pii_detected_response'])

    print(f"PII Detection Analysis (last {days} days):")
    print(f"  Total requests: {total}")
    print(f"  Input PII detected: {input_pii} ({input_pii/total*100:.1f}%)")
    print(f"  Response PII detected: {response_pii} ({response_pii/total*100:.1f}%)")

    return {
        "total": total,
        "input_pii": input_pii,
        "response_pii": response_pii
    }

# Usage
analyze_pii_detection("apt_live_xyz789", days=7)
```

### Cost Tracking by User

Track costs per user:

```python
def track_user_costs(api_key, start_date, end_date):
    """Track costs broken down by user."""
    from collections import defaultdict

    user_stats = defaultdict(lambda: {"requests": 0, "cost": 0.0, "tokens": 0})

    page = 1
    while True:
        response = requests.get(
            "https://api.aptly.dev/v1/logs",
            headers={"Authorization": f"Bearer {api_key}"},
            params={
                "start_date": start_date,
                "end_date": end_date,
                "limit": 100,
                "page": page
            }
        )
        result = response.json()

        for log in result['logs']:
            user = log.get('user_id', 'anonymous')
            user_stats[user]["requests"] += 1
            user_stats[user]["cost"] += log['cost_usd']
            user_stats[user]["tokens"] += log['tokens_input'] + log['tokens_output']

        if page >= result['pagination']['total_pages']:
            break
        page += 1

    # Print report
    print("Cost Breakdown by User:")
    for user, stats in sorted(user_stats.items(), key=lambda x: x[1]['cost'], reverse=True):
        print(f"\n{user}:")
        print(f"  Requests: {stats['requests']:,}")
        print(f"  Tokens: {stats['tokens']:,}")
        print(f"  Cost: ${stats['cost']:.2f}")

    return dict(user_stats)

# Usage
track_user_costs("apt_live_xyz789", "2026-01-01", "2026-01-31")
```

### Export to CSV

Export audit logs to CSV for analysis:

```python
import csv

def export_logs_to_csv(api_key, start_date, end_date, filename="audit_logs.csv"):
    """Export audit logs to CSV."""
    logs = []
    page = 1

    # Fetch all logs
    while True:
        response = requests.get(
            "https://api.aptly.dev/v1/logs",
            headers={"Authorization": f"Bearer {api_key}"},
            params={
                "start_date": start_date,
                "end_date": end_date,
                "limit": 100,
                "page": page
            }
        )
        result = response.json()
        logs.extend(result['logs'])

        if page >= result['pagination']['total_pages']:
            break
        page += 1

    # Write to CSV
    with open(filename, 'w', newline='') as csvfile:
        fieldnames = ['id', 'timestamp', 'model', 'user_id', 'tokens_input',
                     'tokens_output', 'cost_usd', 'duration_ms',
                     'pii_detected_input', 'pii_detected_response', 'status']
        writer = csv.DictWriter(csvfile, fieldnames=fieldnames)

        writer.writeheader()
        for log in logs:
            writer.writerow({k: log.get(k) for k in fieldnames})

    print(f"✓ Exported {len(logs)} logs to {filename}")

# Usage
export_logs_to_csv("apt_live_xyz789", "2026-01-01", "2026-01-31")
```

## Pagination

To fetch all logs across multiple pages:

```python
def fetch_all_logs(api_key, **filters):
    """Fetch all logs with pagination."""
    all_logs = []
    page = 1

    while True:
        response = requests.get(
            "https://api.aptly.dev/v1/logs",
            headers={"Authorization": f"Bearer {api_key}"},
            params={**filters, "page": page, "limit": 100}
        )
        result = response.json()

        all_logs.extend(result['logs'])

        if page >= result['pagination']['total_pages']:
            break

        page += 1

    return all_logs

# Usage
all_logs = fetch_all_logs(
    "apt_live_xyz789",
    start_date="2026-01-01",
    end_date="2026-01-31",
    model="gpt-4"
)
```

## Important Notes

### Immutable Logs

<Note>
  Audit logs are **immutable** - they cannot be modified or deleted. This is enforced by database triggers for compliance.
</Note>

### Default Date Range

If no `start_date` is provided, logs from the **last 30 days** are returned.

### Retention Period

Logs are retained according to your customer profile's `retention_days` setting (default: 2555 days / ~7 years).

### Cost Calculation

The `cost_usd` field is an **estimate** based on token usage and known provider pricing. Actual costs may vary slightly.
