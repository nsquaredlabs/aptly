Get a paginated list of all customers in the system, ordered by creation date (newest first).

## Authentication

<ParamField header="X-Admin-Secret" type="string" required>
  The admin secret configured in your `APTLY_ADMIN_SECRET` environment variable
</ParamField>

## Query Parameters

<ParamField query="limit" type="integer" default={50}>
  Maximum number of customers to return (1-100)
</ParamField>

<ParamField query="offset" type="integer" default={0}>
  Number of customers to skip (for pagination)
</ParamField>

## Pagination

To paginate through all customers:

```python
import requests

def get_all_customers(admin_secret):
    """Fetch all customers using pagination."""
    all_customers = []
    offset = 0
    limit = 50

    while True:
        response = requests.get(
            "https://api.aptly.dev/v1/admin/customers",
            headers={"X-Admin-Secret": admin_secret},
            params={"limit": limit, "offset": offset}
        )
        result = response.json()

        all_customers.extend(result['customers'])

        # Check if we've retrieved all customers
        if offset + limit >= result['total']:
            break

        offset += limit

    return all_customers

# Usage
customers = get_all_customers("your-admin-secret")
print(f"Retrieved {len(customers)} customers")
```

## Use Cases

### Customer Dashboard

Build an admin dashboard showing all customers:

```python
import requests
from datetime import datetime

def build_customer_dashboard(admin_secret):
    response = requests.get(
        "https://api.aptly.dev/v1/admin/customers",
        headers={"X-Admin-Secret": admin_secret},
        params={"limit": 100, "offset": 0}
    )
    result = response.json()

    # Analyze customer base
    total = result['total']
    pro_count = sum(1 for c in result['customers'] if c['plan'] == 'pro')
    free_count = total - pro_count

    print(f"Total Customers: {total}")
    print(f"Pro Plan: {pro_count} ({pro_count/total*100:.1f}%)")
    print(f"Free Plan: {free_count} ({free_count/total*100:.1f}%)")

    # Recent signups
    print("\nRecent Signups:")
    for customer in result['customers'][:5]:
        created = datetime.fromisoformat(customer['created_at'].replace('Z', '+00:00'))
        print(f"  {customer['email']} - {created.strftime('%Y-%m-%d')}")
```

### Export to CSV

Export customer list for reporting:

```python
import requests
import csv

def export_customers_csv(admin_secret, filename="customers.csv"):
    # Get all customers
    all_customers = get_all_customers(admin_secret)

    # Write to CSV
    with open(filename, 'w', newline='') as csvfile:
        fieldnames = ['email', 'company_name', 'plan', 'created_at']
        writer = csv.DictWriter(csvfile, fieldnames=fieldnames)

        writer.writeheader()
        for customer in all_customers:
            writer.writerow({
                'email': customer['email'],
                'company_name': customer.get('company_name', ''),
                'plan': customer['plan'],
                'created_at': customer['created_at']
            })

    print(f"Exported {len(all_customers)} customers to {filename}")
```

### Filter by Plan

Find all customers on a specific plan:

```python
def get_customers_by_plan(admin_secret, plan="pro"):
    """Get all customers on a specific plan."""
    all_customers = get_all_customers(admin_secret)
    filtered = [c for c in all_customers if c['plan'] == plan]

    print(f"Found {len(filtered)} customers on {plan} plan:")
    for customer in filtered:
        print(f"  - {customer['email']}")

    return filtered
```

## Sorting

Customers are always returned in descending order by `created_at` (newest first). This cannot be customized.

## Best Practices

1. **Use appropriate page sizes** - Default 50 is good for most cases
2. **Cache results** - Customer list changes infrequently
3. **Handle pagination** - Don't assume all customers fit in one page
4. **Respect rate limits** - Even admin endpoints have limits
